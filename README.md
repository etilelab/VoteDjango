## Vote Web page + Discord Chatbot
### 개발목표
- Wap 동아리의 투표 불편함을 개선하기 위한 투표 웹 프로젝트
- 디스코드 채팅봇과 투표기능의 융합으로 편의 제공
- 백앤드, 프론트앤드 등 다양한 분야에 대한 협업

### 활용기술
- python3
- django
- sqlite3
- discord lib
- bootstrap
- google Chart

### 개발역할
- Backend (이태경)
- Frontend (길도연)
- Discord Chatbot (정승우)
